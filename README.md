# Australian Housing Insights Dashboard (Power BI)

This project analyses Australian dwelling types, occupancy patterns, and structural distribution using multi-table census data. It includes KPI cards, bar charts, donut charts, and insight summaries.

## Key Features
- KPI cards summarising total, occupied, unoccupied, and non-private dwellings
- Bar chart comparing dwelling types across Australian states
- Donut chart showing dwelling structure distribution
- Insight section summarising key findings

## Tools Used
- Power BI (Web)
- Data modelling
- Visualisation design
- Analytical storytelling
